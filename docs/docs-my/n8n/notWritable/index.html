notWritable

Not Write 出错​

NodeApiError: The file "/home/user/n8n/test.png" is not writable.

打开配置文件

nano ~/.zshrc

添加以下内容

export N8N_BINARY_DATA_MODE=filesystem
export N8N_RESTRICT_FILE_ACCESS_TO=/home/user/n8n

保存后执行

source ~/.zshrc

启动 n8n

n8n
