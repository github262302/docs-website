代理

如果你使用linux遇到了网络问题，可以配置代理。代理链接自己找。本文只推荐GUI程序，不涉及纯命令环境下的代理配置。

  • clash-verge-rev

  • flclash（英文字母小写L）

  • daed

推荐使用flclash。

测试代理是否生效​

配置完代理后一定要测试是否生效。

curl -I www.google.com

返回由HTTP ... OK开头的一大串内容就是成功了

安装临时图形环境​

如果你目前没有桌面，可以安装一个临时的图形化环境运行接下来介绍的软件，我推荐使用labwc。如果你已经有桌面了（任意桌面都可以），不需要这一步。

  1. 安装labwc

    sudo pacman -S labwc kitty

    labwc是一个堆叠式窗口管理器，kitty是我使用的终端。会问你装哪个字体，回车默认就行

  2. 启动labwc

    labwc

    labwc打开之后是纯黑的，正常点击桌面选择terminal就能打开终端，选exit可以退出

  • 卸载labwc

    要做的事情结束之后想删除这个临时图像环境可以使用这条命令：

    sudo pacman -Rns labwc

  2. 安装

  3. 启动

  4. 主页开启TUN（虚拟网卡）

  5. 导入链接

  6. 右下角启动代理

  7. 测试是否生效

clash-verge-rev​

  1. 需要先添加archlinuxcn

  2. 安装

    pacman -S clash-verge-rev

    clash-verge是基于mihomo内核和tauri的面板软件

  3. 启动

    clash-verge

  4. 启动tun模式（虚拟网卡模式）

    如果出现TUN无法安装的情况，可以切换到root身份后打开clash-verge。

    su - 

    clash-verge

    还可以ctrl+alt+f2~f8切换到另一个tty用root身份登录后启动labwc，这样就是root身份开启的图形化环境了。

    如果启用TUN之后没有生效，可以尝试进入设置页面点击clash内核边上的齿轮，切换成mihomo alpha内核，重启内核。

  5. 记得导入链接和启动代理

  6. 测试是否生效

daed​

  1. 安装

    yay -S daed

  2. 启动

    sudo systemctl start daed

  3. webui

    daed的面板以webui方式提供

    打开浏览器，访问localhost:2023即可进入webui。导入订阅之后把订阅从右侧拖到左侧的群组。

  4. 用其他设备访问webui

    运行ip a命令获取本机ip地址。然后打开手机浏览器，在处于同一局域网的情况下访问以下地址：

    你的ip地址:2023

    假设我的ip地址是192.168.0.155，那就用手机浏览器访问192.168.0.155:2023

  5. 开启TUN虚拟网卡

  6. 测试是否生效

v2ray​

yay -S v2rayn-bin

# 右上角修改语言, 主题
# 设置 -> 参数设置 -> v2rayN设置 -> "开机启动", "启动后隐藏窗口", 修改字体
# 订阅分组 -> 添加链接
# 帮助 -> 检查更新
# 可选: 设置 -> 路由设置 -> "一键导入规则集", 再导入自定义配置, 调整顺序
