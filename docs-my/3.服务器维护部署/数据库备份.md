你这条命令思路是对的，但**写反了方向**。

### 更稳的方式（我更推荐）

1. 先压缩 dump：

```powershell
mongodump --host 127.0.0.1:27017 --archive=dump.gz --gzip
```

2. 再导入：

```powershell
mongorestore --archive=dump.gz --gzip
```
```powershell

mongorestore --host "127.0.0.1" --port 27017 -u "your_username" -p "your_password" --authenticationDatabase "admin" --archive=dump.gz --gzip

```
网络不稳定也不会坏。
